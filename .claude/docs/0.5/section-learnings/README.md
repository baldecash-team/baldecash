# Learnings v0.5

Documentación de aprendizajes durante la iteración de la versión 0.5.

## Estructura

Los learnings se organizan por sección:

| Archivo | Sección |
|---------|---------|
| `LEARNINGS_*.md` | Aprendizajes específicos por sección |

---

## Cómo Documentar

Cuando encuentres patrones útiles o soluciones a problemas, documentarlos aquí:

```markdown
## [Fecha] - Título del Aprendizaje

### Contexto
Qué estabas haciendo cuando surgió el aprendizaje.

### Problema
Qué dificultad encontraste.

### Solución
Cómo lo resolviste.

### Aplicación
Dónde más aplica este aprendizaje.
```
